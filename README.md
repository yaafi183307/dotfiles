## ‚öôÔ∏è Installation Instructions

### For macOS Users

1. **Open Terminal**  
   You can find Terminal by searching in Spotlight or navigating to `Applications > Utilities > Terminal`.

2. **Copy the Dotfiles**  
   Move the downloaded file to your home directory using the command:

   ```bash
   mv ~/Downloads/dotfiles-<version>.tar.gz ~/
   ```

3. **Extract the Files**  
   Run the following command to unpack the tar.gz file:

   ```bash
   tar -xzf dotfiles-<version>.tar.gz
   ```

4. **Set Up the Configuration**  
   Navigate to the dotfiles directory:

   ```bash
   cd dotfiles
   ```

   Now run the setup script:

   ```bash
   ./setup.sh
   ```

5. **Enjoy Your New Setup**  
   Close and reopen Terminal to see the changes.
